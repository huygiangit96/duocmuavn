# I18N-8 - Dich layout components dung chung

## Muc tieu

Dich cac component layout dung chung bang `next-intl` va giu locale khi navigate.

## File da cap nhat

### `frontend/src/components/layout/Footer.tsx`

- Import `useTranslations` tu `next-intl`.
- Dung `Link` tu `@/i18n/navigation`.
- Thay cac text chinh bang translation keys:
  - `footer.products`
  - `footer.support`
  - `footer.contact`
  - `footer.copyright`
  - `footer.admin_link`
  - `header.slogan`
- Category links van lay tu API, fallback giu san local.

### `frontend/src/components/layout/CartDrawer.tsx`

- Import `useTranslations` tu `next-intl`.
- Dung `Link` tu `@/i18n/navigation`.
- Thay text bang translation keys:
  - `cart.title`
  - `cart.empty`
  - `cart.shop_now`
  - `cart.checkout`
  - `cart.subtotal`

### `frontend/src/components/Toast.tsx`

- Khong can thay doi.
- Toast tiep tuc hien message tu store.
- Caller se truyen message da dich.

### `frontend/src/components/ProductCard.tsx`

- Import `useTranslations` tu `next-intl`.
- Dung router tu `@/i18n/navigation`.
- Nut them vao gio dung `product.add_to_cart`.
- Toast khi add item dung `cart.added`.

### `frontend/src/components/Pagination.tsx`

- Import `useTranslations` tu `next-intl`.
- Nut truoc/sau dung:
  - `common.prev`
  - `common.next`
- Page number co `aria-label` dung `common.page`.
