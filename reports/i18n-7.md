# I18N-7 - Language switcher, Header va API lang

## Muc tieu

Them language switcher cho frontend, cap nhat Header dung messages cua `next-intl`, va tu dong gui query param `lang` cho cac GET API.

## File da tao

### `frontend/src/i18n/navigation.ts`

- Tao navigation helpers bang `createNavigation(routing)`.
- Export:
  - `Link`
  - `redirect`
  - `usePathname`
  - `useRouter`
  - `getPathname`

### `frontend/src/components/LangSwitcher.tsx`

- Dung `useLocale()` tu `next-intl`.
- Dung `usePathname()` va `useRouter()` tu navigation helper cua `next-intl`.
- Doi locale bang `router.replace(pathname, { locale })`.
- Giu nguyen path hien tai khi chuyen ngon ngu.
- UI gom 2 nut `VI` va `EN`, co active state theo yeu cau.

## File da cap nhat

### `frontend/src/components/layout/Header.tsx`

- Import `useTranslations` tu `next-intl`.
- Thay text hardcode bang key messages:
  - `header.hotline`
  - `header.slogan`
  - `nav.home`
  - `nav.products`
  - `nav.news`
  - `nav.docs`
  - `nav.contact`
  - `header.search_placeholder`
  - `common.search`
  - `auth.login`
  - `header.account`
  - `header.cart`
- Them `<LangSwitcher />` vao hang icon ben phai, sau icon user va truoc hamburger.
- Chuyen Link/router/pathname sang navigation helper cua `next-intl` de link tu dong giu locale.

### `frontend/messages/vi.json`

- Them `header.slogan`.

### `frontend/messages/en.json`

- Them `header.slogan`.

### `frontend/src/lib/api.ts`

- Request interceptor tu dong them `lang=<locale>` cho moi GET request.
- Doc locale tu cookie `NEXT_LOCALE`.
- Mac dinh `vi` neu khong co cookie.
- Khong them `lang` cho POST/PUT/PATCH/DELETE.
- Neu GET da co `lang` thi giu gia tri hien co.

### `frontend/src/components/layout/Providers.tsx`

- Dieu chinh check admin path de ho tro ca `/quan-tri` va `/en/quan-tri`.
