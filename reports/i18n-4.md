# I18N-4 - Admin API CRUD song ngu

## Muc tieu

Cap nhat Admin API serializers de ho tro CRUD cac field tieng Anh (`*_en`) va cho phep admin de trong cac field nay.

## Serializer da cap nhat

File: `backend/core/admin_serializers.py`

- `AdminProductSerializer`
  - `name_en`
  - `short_desc_en`
  - `description_en`
  - `specs_en`
- `AdminCategorySerializer`
  - `name_en`
  - `description_en`
- `AdminPlantSerializer`
  - `name_en`
- `AdminArticleSerializer`
  - `title_en`
  - `summary_en`
  - `content_en`
- `AdminDocumentSerializer`
  - `title_en`
  - `summary_en`
  - `content_en`
- `AdminNewsCategorySerializer`
  - `name_en`
- `AdminDocCategorySerializer`
  - `name_en`
- `AdminBannerSerializer`
  - `title_en`
  - `subtitle_en`
- `AdminSiteConfigSerializer`
  - `tagline_en`
  - `about_en`

## Cach khai bao

- Cac field text dung `serializers.CharField(required=False, allow_blank=True)`.
- Field JSON `specs_en` dung `serializers.JSONField(required=False)`.
- Cac serializer van giu `fields = "__all__"` de khong lam mat field hien co cua Admin API.
